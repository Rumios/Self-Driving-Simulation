## LiDAR 센서 기반 자율주행 시스템

본 프로젝트는 **2D LiDAR 센서를 활용한 자율주행 시스템**을 구현하는 것을 목표로 하며, **Deep Q-Network(DQN)** 알고리즘을 이용하여 차량이 주변 환경을 인식하고 목적지까지 스스로 주행할 수 있도록 설계하였다.

차량은 일정 시야각(Field of View) 내에서 **Raycasting**을 수행하여 주변 장애물과의 거리를 실시간으로 측정한다. 각 Ray는 최대 탐지 거리까지 전진하며 장애물과 충돌하는 지점을 탐색하고, 충돌 시 해당 거리 정보를 획득하여 **Point Cloud** 데이터로 변환한다.

생성된 Point Cloud는 차량 기준의 **Local 좌표계**와 실제 환경을 나타내는 **Global 좌표계** 사이의 좌표 변환을 거쳐 지도에 누적되며, 이를 통해 주변 환경을 지속적으로 인식하고 맵을 구축한다. 또한 최대 탐지 거리까지 아무것도 감지되지 않은 데이터는 제외하여 불필요한 정보를 최소화하고, 실제 장애물 정보만을 활용하여 환경을 표현한다.

최종적으로 DQN은 LiDAR 센서로부터 획득한 주변 환경 정보와 차량의 현재 상태를 입력으로 받아 가속, 감속, 조향 등의 행동을 학습하며, 장애물을 회피하고 교통 환경을 고려하여 목표 지점까지 안전하게 이동하는 자율주행 정책을 학습하도록 구성하였다.
